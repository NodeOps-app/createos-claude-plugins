# createos — Claude Code plugin marketplace

NodeOps [CreateOS](https://createos.sh) plugins for Claude Code — disposable sandbox compute and live dev sessions.

## Plugins

| Plugin | What |
|---|---|
| [`createos-sandbox`](./createos-sandbox) | Run ad-hoc / heavy / untrusted code in disposable CreateOS microVMs: offload, parallel fanout, scratch shell, reusable box with sync, port tunnel, public expose, network clusters, S3 disks, WireGuard VPN, and fork. |

## Install

```
/plugin marketplace add /path/to/createos
/plugin install createos-sandbox@createos
```

See each plugin's own README for usage.
